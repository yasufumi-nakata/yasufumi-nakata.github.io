[What is SKIP?]

SKIPis an application to put information sharing based on communications into practice.

The main targets are knowledge sharing and promoting communications.
It provides features designed to enable fun and meaningful communications between users including job-related know-hows and even little chit-chats.

Every SKIP user owns a personal page (called "My page") so that everyone can publish and collect information.
One can write down and publish daily thoughts and know-hows acquired in relation with their jobs as blogs and search for knowledge and information published by other users.

SKIP also allows users to create groups.
Users can find buddies with the same target and interests by joining groups and make more meaningful commucications and accumulate know-hows. They can also create groups by themselves.

For more detailed explanations for the features and the usage, please refer to the help documentations.
There are lots of fun and useful features made ready for you to communicate and discover people, knowledge and information.
So come and have great fun!
